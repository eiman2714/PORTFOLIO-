# 🌸 Eiman Waheed | Pastel Portfolio

A beautiful, girly pastel-themed portfolio website designed for Eiman Waheed - a Computer Science student and Front-End Developer intern at Optimus Automate. This portfolio showcases skills, projects, and professional information with a soft, feminine aesthetic.

---

## ✨ Features

### 🎨 Design & Visuals
- **Pastel Color Palette** - Soft pinks, whites, and gentle tones for a feminine, elegant look
- **Custom Cursor** - Unique pink-themed cursor that follows mouse movements
- **Animated Particles** - Floating soft pink particles in the hero section
- **Glass-morphism Cards** - Modern, frosted glass effect on all content cards
- **Gradient Text** - Beautiful animated gradient on the main name heading
- **Typing Animation** - Dynamic text that cycles through different role descriptions
- **Responsive Design** - Looks perfect on all devices from mobile to desktop

### 🎯 Interactive Features
- **Loading Screen** - Smooth loading animation when page first opens
- **Navigation Menu** - Sticky navbar with smooth scrolling and mobile hamburger menu
- **Skills Progress Bars** - Animated skill indicators that load when you scroll
- **Calculator Modal** - Fully functional calculator with basic operations
- **Music Player Modal** - Interactive music player with play/pause, volume control, and track switching
- **Contact Form** - Working contact form with validation and success message

### 📋 Content Sections
1. **Hero Section** - Introduction with animated typing effect and call-to-action buttons
2. **About Me** - Detailed professional information with education and experience
3. **Skills** - Visual representation of technical proficiency with percentage bars
4. **Projects** - 5 project cards with demo and GitHub links
5. **Contact** - Contact information, form, and social media links

---

## 🛠️ Technologies Used

### Frontend
- **HTML5** - Semantic, accessible markup
- **CSS3** - Custom styles with animations, flexbox, grid, and glass-morphism
- **JavaScript** - Vanilla JS for interactivity, animations, and modals

### Libraries & Fonts
- **Font Awesome 6** - Icon library for social media and UI icons
- **Google Fonts (Quicksand)** - Soft, elegant font for the entire site


## 📱 Responsive Design

The portfolio is fully responsive and looks great on:

- 💻 **Desktop** - Full experience with custom cursor and animations
- 📱 **Tablet** - Adapted layout for medium screens
- 📲 **Mobile** - Collapsed navigation, optimized spacing and fonts
